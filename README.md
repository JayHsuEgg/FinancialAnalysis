## 🚀 GitHub Pages 設置指南

### 方法一：使用 GitHub Pages（推薦）

1. **啟用 GitHub Pages**
   - 前往倉庫的 `Settings` → `Pages`
   - 在 `Source` 選擇 `Deploy from a branch`
   - 選擇 `main`（或 `master`）分支
   - 選擇 `/ (root)` 目錄
   - 點擊 `Save`

2. **訪問網站**
   - 幾分鐘後，網站將在 `https://[你的用戶名].github.io/[倉庫名]/` 上線
   - `index.html` 將自動成為首頁

3. **文件說明**
   - `.nojekyll` 文件已創建，確保 GitHub Pages 正確顯示 HTML 文件
   - 所有 `.md` 文件可以直接在 GitHub 上查看
   - `index.html` 提供完整的導航目錄

### 方法二：使用 GitHub 原生 Markdown 渲染

如果只想在 GitHub 上查看，所有 `.md` 文件已經可以直接瀏覽。
