## 🎯 專案目標

本專案收錄 2025-2026 年長期指數化投資策略研究，核心理念基於學術論文 *"Beyond the Status Quo: A Critical Assessment of Lifecycle Investment Advice"* (Anarkulova, Cederburg, O'Doherty；本專案以 **2024/11/18 修訂版**全文核校，SSRN 4590406)，挑戰傳統生命週期投資建議。

### 核心發現

| 傳統建議 | 學術研究結論 |
|---------|-------------|
| 股債配置 60/40 | ❌ 次優策略 |
| 隨年齡降低股票比例 | ❌ 降低終身財富 |
| 本國偏好 | ❌ 犧牲分散效益 |
| **最佳策略** | **100% 股票（1/3 本國 + 2/3 國際）** |

### 論文關鍵數據（2024/11/18 修訂版核校）

- **資料**：39 個已開發國家、1890–2023、2,600+「國家-年」；基準風險趨避 γ = 3.84。
- **最優配置**：33% 國內股 + 67% 國際股 + 0% 債，**所有年齡不變（終生全股，退休亦然）**。
- **退休破產機率**（4% 提領、身故前歸零，愈低愈好）：最優全股 **7.0%** vs 傳統 TDF 19.7%；長壽情境最優仍僅 14.9%。
- **等效儲蓄率**：達同等效用，TDF 需 16.1%（**多存 61%**）；全股基準較省。
- **槓桿（Panel L）**：最優槓桿取決於借貸利差——散戶利差 6.5% → **最優槓桿 0%**；機構近無風險利率借款 → 100%+。

> ⚠️ 網路流傳的「相信美國例外論 0/50/100% → 權重」對照表在 2024/11/18 版**不存在**，不可引用。

| 平台 | 用途 | 手續費 |
|---|---|---|

- **稅務要點**：美股股息對台灣人（NRA）預扣 **30%**、愛爾蘭 UCITS 基金層僅 **15%**；美國 situs 資產 >$60k 超出部分課 **18–40%** 美國遺產稅，愛爾蘭 UCITS 豁免。
- **SGOV 特例**：美國公債利息依 IRC 871(i) 對 NRA **免預扣**（Firstrade 實測 0%）。

## 💡 投資理念

```
📌 100% 股票配置（無固定收益）
📌 1/3 台股（0050）+ 2/3 國際（VWRA）
📌 短債作為抄底彈藥庫，非長期配置
📌 定期定額 + 逢低加碼
📌 長期持有，忽略短期波動
```
