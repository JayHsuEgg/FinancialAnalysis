# 📈 長期指數化投資研究專案

> 基於學術研究的被動投資策略分析與執行計畫

[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

## 🎯 專案目標

本專案收錄 2025-2026 年長期指數化投資策略研究，核心理念基於學術論文 *"Beyond the Status Quo: A Critical Assessment of Lifecycle Investment Advice"* (Anarkulova, Cederburg, O'Doherty, 2023)，挑戰傳統生命週期投資建議。

### 核心發現

| 傳統建議 | 學術研究結論 |
|---------|-------------|
| 股債配置 60/40 | ❌ 次優策略 |
| 隨年齡降低股票比例 | ❌ 降低終身財富 |
| 本國偏好 | ❌ 犧牲分散效益 |
| **最佳策略** | **100% 股票（1/3 本國 + 2/3 國際）** |

## 💡 投資理念

```
📌 100% 股票配置（無固定收益）
📌 1/3 台股（0050）+ 2/3 國際（VWRA）
📌 短債作為抄底彈藥庫，非長期配置
📌 定期定額 + 逢低加碼
📌 長期持有，忽略短期波動
```
