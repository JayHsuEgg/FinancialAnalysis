# 短期美債ETF完整分析：SGOV vs IB01 vs BOXX

---

## 執行摘要

**核心結論：依券商與資產規模選擇最適工具**

| 情境 | 推薦 | 原因 |
|-----|------|------|
| **IB 開戶 + 資產 > 6萬美元** | **IB01** | 避遺產稅 + 累積型 + 低費率 |
| **Firstrade + 資產 < 6萬美元** | **BOXX** | 無配息免預扣稅問題 + 高報酬 |
| **Firstrade + 資產 > 6萬美元** | **IB01** (需換券商) | 遺產稅風險 > 操作便利 |
| **需要月現金流** | **SGOV** | 但須注意券商預扣稅處理 |

> ⚠️ **重要提醒**：部分券商（如 Firstrade）會對 SGOV 月配息預扣 30%，即使理論上美國國債利息免稅。若無法確認退稅機制，應優先選擇累積型 ETF（IB01 或 BOXX）。

### 投資範圍影響分析

**SGOV 的超短期特性：**
- **優勢**：幾乎零利率風險，價格極其穩定
- **劣勢**：收益率通常較低，周轉成本較高，月配息有預扣稅風險
- **適用**：現金替代品，短期資金停泊

**IB01 的短期特性：**
- **優勢**：收益率稍高，累積型設計，愛爾蘭註冊避遺產稅
- **劣勢**：輕微利率敏感度，流動性較低
- **適用**：長期短債配置，重視遺產規劃者

**BOXX 的期權策略：**
- **優勢**：收益為資本利得（非利息），無配息免預扣稅，報酬最高
- **劣勢**：流動性最低，策略複雜，美國註冊有遺產稅風險
- **適用**：Firstrade 用戶、不需月現金流、資產 < 6萬美元

## BOXX 特別分析：箱型價差策略

### 什麼是 BOXX？

**Alpha Architect 1-3 Month Box ETF (BOXX)** 使用 S&P 500 箱型價差期權策略來模擬短期國債報酬，但收益性質為**資本利得**而非利息。

### BOXX 的核心優勢

| 優勢 | 說明 |
|-----|------|
| **無配息** | 累積型設計，完全避免預扣稅問題 |
| **收益為資本利得** | 非美稅務居民避免利息 30% 預扣稅爭議 |
| **報酬較高** | 年化 ~4.8% vs SGOV ~4.2%，多 0.6% |
| **稅務簡化** | 無需每月處理配息申報 |
| **操作零風險** | 不需盯配息日賣出 |

### BOXX 的風險與劣勢

| 劣勢 | 說明 |
|-----|------|
| **流動性較低** | 日均交易量僅 SGOV 的 7%，大額進出有滑價風險 |
| **費率較高** | 0.19% vs SGOV 0.09%，多 0.1% |
| **策略複雜** | 期權價差非傳統國債持有，需理解策略 |
| **遺產稅風險** | 美國註冊，資產 > 6萬美元面臨 40% 遺產稅 |
| **規模較小** | $74億，相對基金清算風險較高 |

### BOXX vs SGOV 實際報酬比較

| 項目 | SGOV | BOXX | 差異 | 說明 |

| 年化總報酬（已扣內扣費用）| 4.22% | 4.80% | +0.58% | ETF 公布數字已含費用 |
| Firstrade 預扣稅損失 | -1.26%* | 0% | +1.26% | 30% × 4.22% |
| **實際淨報酬** | **2.96%** | **4.80%** | **+1.84%** | - |

## 券商預扣稅實務問題

### ⚠️ Firstrade 等券商的預扣稅陷阱

**理論 vs 現實：**
- **理論上**：美國國債利息依 IRC 871(i)(2)(A) 對非美居民免稅
- **現實上**：部分券商（如 Firstrade）仍會對 SGOV 月配息預扣 30%
- **退稅不確定**：Firstrade 未明文承諾返還預扣稅

**問題嚴重性：**

| 情境 | 影響 |
|-----|------|
| 每月被預扣 30% 配息 | 年損失約 1.26%（4.2% × 30%）|
| 退稅不確定 | 現金流被鎖住一整年 |
| 需每月配息前賣出 | 操作風險 + 時間成本 |
| 漏賣一次 | 損失該月 ~0.35% |

### 解決方案比較

| 方案 | 操作 | 風險 | 推薦度 |
|-----|------|------|--------|
| **換用 BOXX** | 買入後持有，無需任何操作 | 低 | ⭐⭐⭐⭐⭐ |
| **換用 IB01** | 需開 IB 帳戶，買入後持有 | 低 | ⭐⭐⭐⭐⭐ |
| **每月賣出 SGOV** | 配息前賣、配息後買 | 高（漏賣風險） | ⭐⭐ |
| **申請退稅** | 年底申請，不確定性高 | 中 | ⭐⭐⭐ |

### 主力策略建議

**情境一：IB 開戶 + 長期投資**
- **配置**：IB01 100%
- **理由**：避遺產稅 + 累積型 + 最低費率 + 無預扣稅問題

**情境四：需要月現金流**
- **配置**：SGOV（謹慎使用）
- **注意**：需每月配息日前賣出，配息後買回，避免被預扣 30%

## 結論

### 核心差異總結

| 維度 | 最佳選擇 | 原因 |
|-----|---------|------|
| **遺產稅規劃** | IB01 | 愛爾蘭註冊，完全避免 |
| **預扣稅問題** | BOXX/IB01 | 累積型無配息 |
| **報酬最大化** | BOXX | 年化 ~4.8%，最高 |
| **流動性最高** | SGOV | 日均交易量最大 |
| **費用最低** | IB01 | 0.07% |
| **Firstrade 用戶** | BOXX | 無預扣稅風險 |

### 關鍵數據總結

- **BOXX vs SGOV 年報酬差**：+0.58%（不計預扣稅）
- **BOXX vs SGOV 年報酬差**：+1.74%（計入 Firstrade 預扣稅）
- **IB01 遺產稅節省**：資產的 40%
- **BOXX 流動性**：僅 SGOV 的 7%

### 最終推薦

對於使用 Firstrade 且資產 < 6萬美元的台灣投資者：

**BOXX 是最佳選擇**

1. ✅ 無配息 = 無預扣稅問題
2. ✅ 不需每月盯配息日
3. ✅ 報酬最高 (~4.8%)
4. ✅ 操作風險歸零
5. ⚠️ 唯一注意：資產成長超過 6 萬美元後，應考慮開 IB 帳戶轉用 IB01
