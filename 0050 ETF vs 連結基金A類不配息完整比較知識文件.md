## 二、內扣費用詳細計算

### 0050 ETF費用結構（規模6,473.76億）

#### 經理費計算
```
前1,000億：0.15%
1,000-5,000億：0.10%  
5,000億-1兆：0.08%
超過1兆：0.05%

實際計算：
= (1,000億×0.15% + 4,000億×0.10% + 1,473.76億×0.08%) ÷ 6,473.76億
= (1.5億 + 4.0億 + 1.179億) ÷ 6,473.76億
= 0.103%
```

#### 保管費
```
1兆以下：0.03%
```

#### 0050總費用率
```
經理費 + 保管費 = 0.103% + 0.03% = 0.133%
```

### 連結基金A類費用結構（規模240億）

#### 經理費計算
```
投資0050部分（95%）：不收取經理費
現金部位（5%）：按0050經理費規則收取
- 0050經理費規則：1,000億以下部分為0.15%
- 實際計算：5% × 0.15% = 0.0075%
```

#### 保管費計算
```
前50億：0.02%
50-100億：0.015%
100億以上：0.01%

實際計算：
= (50億×0.02% + 50億×0.015% + 140億×0.01%) ÷ 240億
= (0.01億 + 0.0075億 + 0.014億) ÷ 240億
= 0.01313%
```

#### 承擔0050費用
```
95% × 0.133% = 0.1264%
```

#### 連結基金總費用率
```
經理費 + 保管費 + 0050費用
= 0.0075% + 0.01313% + 0.1264%
= 0.14703%
```

### 費用差異分析
- 0050 ETF：0.133%
- 連結基金A類：0.14703%
- 差異：0.01403%（每萬元每年多1.4元）
